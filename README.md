# HTTP 接口管理系统

一个现代化的 HTTP/REST API 接口管理系统，提供完整的接口注册、管理和测试功能。

## 功能特性

### 🔌 HTTP协议支持
- **HTTP/REST API** - 支持 GET、POST、PUT、DELETE、PATCH 等方法
- **灵活参数位置** - 支持URL查询参数和请求体两种参数位置
- **多种认证方式** - 支持 Bearer Token、Basic Auth、API Key 等认证方式

### 📋 接口管理
- **接口注册** - 添加和配置不同协议的接口
- **CRUD 操作** - 完整的创建、读取、更新、删除功能
- **接口列表** - 清晰的界面展示所有已注册接口
- **搜索过滤** - 按名称、协议类型、状态等条件筛选



### ⚙️ 接口配置
- **认证支持** - Bearer Token、Basic Auth、API Key 等
- **参数配置** - 灵活的参数类型和验证
- **接口测试** - 内置接口测试功能

## 快速开始

1. 打开 `index.html` 文件
2. 系统已预置了一些示例接口数据
3. 点击"添加接口"开始创建您的接口

## 使用说明

### 添加接口

1. 点击侧边栏的"添加接口"
2. 填写基本信息：
   - 接口名称（必填）
   - 描述
   - 协议类型（必填）
   - HTTP 方法（GET、POST、PUT、DELETE、PATCH）
   - 参数位置（可选择URL查询参数或请求体）
3. 配置连接信息：
   - 端点 URL（必填）
   - 认证类型和凭据
4. 添加请求参数：
   - 参数名、类型、是否必填、描述
5. 点击"保存接口"

### 接口测试

1. 在接口列表中点击"测试"按钮
2. 填写必要的参数
3. 查看接口响应结果

## 技术特点

- **响应式设计** - 适配桌面和移动设备
- **现代化 UI** - 简洁直观的用户界面
- **数据持久化** - 本地存储接口配置
- **类型安全** - 完整的参数类型验证
- **接口测试** - 内置HTTP请求测试功能
