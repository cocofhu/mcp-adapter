## 🔧 技术实现

### 1. 无框架设计
- 使用原生 JavaScript（ES6+）
- 无需构建工具
- 轻量级，快速加载
- 易于维护和扩展

### 2. API 集成
```javascript
// 统一的 API 请求封装
async function apiRequest(url, options = {}) {
    // 自动添加 Content-Type
    // 统一错误处理
    // 支持 JSON 响应
}
```

### 3. 状态管理
```javascript
const state = {
    currentApp: null,      // 当前选中的应用
    applications: [],      // 应用列表
    customTypes: [],       // 自定义类型列表
    interfaces: []         // 接口列表
};
```

### 4. 模态框系统
```javascript
showModal(title, bodyHTML, onConfirm, showFooter)
// 灵活的模态框显示
// 支持自定义内容
// 支持确认回调
```

### 5. Toast 通知
```javascript
showToast(message, type)
// 类型：success, error, warning, info
// 自动消失（3秒）
// 动画效果
```

## 🔄 与后端集成

### API 端点映射
```
前端功能              →  后端 API
─────────────────────────────────────
应用管理              →  /api/applications
自定义类型管理        →  /api/custom-types
接口管理              →  /api/interfaces
```

### 数据流
```
用户操作 → 前端验证 → API 请求 → 后端处理 → 响应 → 更新 UI → Toast 通知
```
