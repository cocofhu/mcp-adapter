自定义类型

自定义类型允许您定义可复用的复杂数据结构，类似于 TypeScript 的 interface。

示例：

{
  "name": "User",
  "fields": [
    {"name": "id", "type": "number", "required": true},
    {"name": "name", "type": "string", "required": true},
    {"name": "email", "type": "string", "required": false}
  ]
}

参数类型

  • number - 数字类型
  • string - 字符串类型
  • boolean - 布尔类型
  • custom - 自定义类型（引用已定义的类型）

参数位置

  • query - URL 查询参数 (?key=value)
  • header - HTTP 请求头
  • body - 请求体（JSON）
  • path - URL 路径参数 (/users/{id})
